# zho-ued-resource

这是一个用于归档和发布多个独立 HTML 页面的仓库。仓库内包含一系列与用户体验设计（UED）相关的静态页面资源，旨在提供可参考、可复用的前端页面存档。

## 项目简介

本仓库收集并整理了一些独立的 HTML 页面，这些页面不针对任何特定域名或具体网站，而是作为通用性设计素材或功能演示存在。页面内容涵盖常见的 UI 组件、交互效果、布局模板等，适合设计师、前端开发者或产品经理参考与学习。

## 目录说明

```
zho-ued-resource/
├── pages/          # 存放所有独立 HTML 页面
├── assets/         # 页面依赖的静态资源（如 CSS、JS、图片等）
├── docs/           # 相关文档或说明文件
└── README.md       # 本文件
```

- `pages/`：每个子文件夹或文件代表一个独立页面，命名清晰，便于索引。
- `assets/`：按需存放页面所需的样式、脚本、字体或图标等资源。
- `docs/`：可选，用于存放页面使用说明、设计思路或更新日志。

## 页面归档说明

仓库中的页面均为独立归档，每个页面可单独打开运行，不依赖外部服务器或数据库。页面可能包含：

- 常见的用户界面组件（如导航、表单、卡片、弹窗等）
- 交互效果演示（如动画、过渡、响应式布局等）
- 设计风格参考（如配色、排版、间距等）

页面内容来源包括但不限于：个人练习、设计探索、开源项目借鉴等。所有资源均遵循相关开源协议，如有引用会在页面内注明。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接使用浏览器打开 `pages/` 下的任意 HTML 文件即可查看效果。
3. 可根据需要复制、修改页面代码用于个人或项目学习。
